Описание----------------------------------------------------------
В этом проекте было реализовано :
- регистрация и
 -авторизация пользовотеля-


 Установка---------------------------------------------------------
 интерпритатор - python 3.11.2

 Нужно в проекте
 создать виртуальное окружение и скачать django
 pip install django



 Использование-----------------------------------------------------------

в терминале зайти на папку в необис в проекте
в папке --ваш-диск/Папка где сохранен этот проект/startproject\Neobis>

запустить -python manage.py runserver

есть такие ссылки как:

''-главная страница

 'about/' -тестовая страница использованием бд





 ------------------реализация регистрации и авторизации с использовнием классов----------------------------

 'accounts/register/'-страница регистрации и авторизации использованием -class RegisterUser,с успотльзоваем- UserCreationForm

 'secondregistr/'-регистрация но реализовано по другому с использованием -RegisterForm

 'secondlogin/'-логин отдельно от регистраци с использованием -AuthenticationForm



--------------реализация регистрации и авторизации с использовнием функции------------
